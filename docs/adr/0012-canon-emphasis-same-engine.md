# ADR 0012 — 캐논/비캐논: 동일 엔진 + 캐논 강조 가산 레이어

## 맥락
들어오는 곡이 캐논이 아닐 수 있다(화음/기타). 레퍼런스는 캐논 전용 연출이지만, 본 앱은 임의의
MIDI를 받는다. 분류별로 시각화 엔진을 따로 만들면 유지보수가 갈라진다.

5초 프레임 관찰로 확인된 캐논 강조 방식 2종:
- **시차 추격(영상1)**: 같은 구조를 여러 마커가 시간차로 따라감(빨강→초록→파랑 시차 출발).
- **거울 대칭(영상2)**: 흰 원본 구조 + 검은 거울상을 동시에, 역행/전위를 기하 반사로.

## 결정
- **단일 엔진(동일 엔진).** 분류(캐논/화음/기타)와 무관하게 코어는 동일:
  "성부별 지형(음높이=높이) + 마커 주행 + ADR 0011 토글". 엔진을 분기하지 않는다.
- **캐논 강조는 코어 위 가산 레이어(옵션).** 캐논으로 분류된 곡에서만 켜지는 추가 표현:
  - **시차 추격 강조**: 성부 간 모방 lag를 분석값으로 받아, 마커 출발/주행을 같은 트랙 위에
    시차 정렬해 "추격"이 드러나게.
  - **거울 대칭 강조(가능 시)**: 역행/전위 관계가 감지되면 거울상 구조를 보조로 표시.
  - 둘 다 켜짐 기본(grill 확정), 끌 수 있음.
- **비캐논(화음/기타)**: 같은 엔진 그대로. 강조 레이어만 꺼진다. 별도 모드 분기 없음.
  - 화음: 여러 성부가 각자 지형을 동시 주행(현행과 동일, 자연스럽게 다성 표현).
  - 기타(단성): 마커 1개가 자기 지형을 주행.

## 근거
스탠스가 분류와 무관하게 성립하므로 엔진을 하나로 유지하는 게 단순·견고하다. 캐논 특유의
모방 관계만 "있으면 더 보여주는" 가산 레이어로 두면, 비캐논 곡도 깨지지 않고 캐논 곡은 풍부해진다.

## 대안
- 분류별 엔진 분기: 유지보수 분기·코드 중복 → 기각(동일 엔진 + 가산 레이어).
- 캐논 강조 상시 적용: 비캐논에 모방 lag가 없어 오작동 → 분류 게이트로 제한.

## 의존
- 시차 추격: 백엔드가 성부쌍 모방 lag를 분석값으로 노출해야 함(현재 `classifier`의 lag 유사도 활용).
- 거울 대칭: 역행/전위 감지는 추가 분석 필요(후순위, 가능 시).
- 분류 오판(캐논곡이 "화음"으로) 보정은 ADR 0005 개정 과제와 연동.
