# ADR 0007 — 입력 방식·MIDI 자동획득

## 맥락
오디오 채보(AMT)는 어려움. 대안: 곡의 기존 MIDI를 자동으로 찾아오면 정확한 음표를 공짜로 얻음.

## 결정
- **입력 방식**: ① 직접 파일 업로드(MVP) + ② 곡이름 텍스트 검색 → 공개 MIDI DB(bitmidi 등)
  검색·후보 제시·다운로드·분석.
- **검색 입력**: 텍스트 검색어(예: "pachelbel canon").
- **시점**: 자동획득은 MVP 이후 Phase 1.5 별도 단계(외부 사이트 의존·저작권 리스크 분리).

## 근거
AMT 없이 정확한 음표 확보. 업로드 병행으로 검색에 없는 곡·내 파일도 처리.

## 대안
- YouTube 링크 / 오디오 지문 식별: 제목 매칭 부정확·비용 큼 → 텍스트 검색 우선.
- MVP에 포함: 외부 의존·불안정 → Phase 1.5로 분리.
