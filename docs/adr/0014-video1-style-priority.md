# ADR 0014 — 영상1(파헬벨) 스타일 우선, 영상2(바흐) 부차

## 상태
확정됨 (2026-06-23)

## 배경
레퍼런스 영상이 두 편이다.
- 영상1: NHK 파헬벨 캐논 — 무채색 디오라마 섬들, 큐브가 시차 통과, 이산 블록 지형
- 영상2: 바흐 14 Canons BWV 1087 — 거울 반사 3평면, 주제 타이틀 씬, 발광 효과

Phase C에서 구현된 거울 대칭(C4)과 Phase G에서 계획된 G2·G3는 영상2 스타일을 따른다.
grill-with-docs 세션에서 사용자가 영상1 스타일을 명시적으로 우선순위로 지정했다.

## 결정
구현 우선순위는 영상1 스타일(파헬벨)이다.

- **영상1 핵심**: 디오라마 섬 위를 큐브가 시차 통과 → 캐논의 본질이 직접 보임.
  이것이 Phase H(공유 섬 + 순환 횡단)의 주된 동기다.
- **영상2 요소**(거울, 3평면, 타이틀 씬)는 구현됐거나(C4) 계획됐지만(G2·G3),
  영상1 동등 수준이 달성된 이후에 추가하는 부차 목표다.

## 결과
- Phase H(영상1 핵심 구현)가 Phase G(영상2 고도화)보다 높은 우선순위를 가진다.
- G2(직교 3평면 반사)·G3(주제 타이틀 오프닝)은 H 완료 후 검토.
- C4(거울 대칭)는 이미 충분히 구현됨 — 추가 투자 불필요.
