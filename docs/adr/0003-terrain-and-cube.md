# ADR 0003 — 지형·큐브·카메라

## 맥락
다성 음악은 동시에 여러 음이 울린다. "큐브가 지형 위를 미끄러진다"는 본질적으로 하나의 선.

## 결정
- **큐브 추적**: 성부마다 큐브 1개 + 성부별 레인.
- **레인 배치**: 깊이(z축)로 나란히, 높은 성부를 앞에.
- **지형 모양**: 음높이 점을 보간한 부드러운 리본/언덕. 큐브가 경사를 따라 미끄러짐.
- **높이 정규화**: 곡 전체 최저~최고음(min/max midi)으로 자동 스케일.
- **화음 처리**: 한 성부 안 화음은 최고음(멜로디)을 큐브 높이로. 화음 전체는 색/지형 보조.
- **쉼표**: 직전 음높이 유지·평지로 미끄러짐(큐브 끊김 없음).
- **음길이(몇분음표)**: 큐브 속도 일정 → 음 길이가 지형 구간 거리로 표현.
- **카메라**: 3인칭 추적(주성부=최고음 성부 큐브 기준).

## 근거
다성 충실 + NHK/바흐 영상의 "여러 줄 동시 흐름" 재현. 곡 전체 음역 자동 스케일로 어떤 곡이든 적당한 기복.

## 대안
- 최상성 1개만 / 전체 한 줄 합성: 음악적 의미 손실 → 기각.
- 고정 음역(0~127): 음역 좁은 곡이 밋밋 → 자동 스케일 채택.
