# ADR 0011 — 레퍼런스 "느낌" 재현 (기존 스탠스 유지 + 토글 비교)

## 맥락
레퍼런스 영상 2편(NHK 명곡앨범플러스: 파헬벨 캐논, 바흐 14 canons)을 직접 다운로드해 5초 간격
프레임으로 관찰했다(→ `docs/reference-videos-digest.md`, `docs/reference-frames/`).

결론: **패러다임은 우리와 같다.** 레퍼런스도 본질은 "음높이=높이, 큐브(마커)가 음표를 따라 주행,
음길이=진행 거리"다. 무채색은 캐논 연출의 부수효과일 뿐 핵심이 아니다. 따라서 기존 스탠스
(`terrain.js`의 지형=음높이, 큐브 주행)를 **버리지 않는다.**

차이는 미감 디테일 4가지에 있고, 이것이 "느낌"의 정체다. 각 항목은 **기존/변경을 나란히 비교**할
수 있도록 토글 옵션으로 제공한다(취향·검증 목적, ADR 0009의 옵션화 원칙 계승).

## 결정 — 차이 포인트 4종을 각각 토글 옵션으로

1. **① 이산 블록화 (terrainShape: smooth | stepped)** ← 가장 큰 효과
   - 현재 `_sampleHeights`의 이동평균(스무딩)이 음표를 언덕으로 뭉갠다.
   - `stepped` 모드: 스무딩 제거, 음표를 **양자화된 계단/블록**(음높이=높이, 음길이=가로폭)으로.
     음표가 개별로 읽힌다. 기존 스탠스와 정확히 일치.
   - 기본 비교용으로 `smooth`(기존)도 유지.

2. **② 발광 제거 (renderStyle: glow | matte)**
   - `glow`(기존): emissive + UnrealBloom(0.9) + 어두운 배경.
   - `matte`(신규): 블룸 거의 끔, **무광 재질 + soft shadow + 중립(밝은 회색) 배경**.
   - 성부색은 둘 다에서 유지(레퍼런스의 무채색은 "마커 외 중립"이라는 규칙일 뿐, 색 자체는 옵션).

3. **③ 디오라마 무대화 (stage: scroll | diorama)**
   - `scroll`(기존): 무한 저공 스크롤 + 추격 카메라.
   - `diorama`(신규): 곡을 **구간별 경계 있는 정사각 "섬"**으로 나누고 길로 연결, **아이소메트릭
     부감**으로 조망. (구간 분할 기준은 구현 시 결정 — 마디/프레이즈/일정 길이)

4. **④ 트랙 가독성 (trackWidth: wide | narrow)**
   - `wide`(기존): 넓은 반투명 지형(벽처럼 보임).
   - `narrow`(신규): 좁고 불투명한 트랙 → "큐브가 달리는 길"로 읽힘.

- **공통 원칙**: 네 토글은 독립적. 전부 기존값이면 현행과 동일, 전부 신규값이면 레퍼런스 느낌.
  사용자가 섞어 보며 무엇이 "그 느낌"을 만드는지 직접 확인한다.

## 근거
스탠스(음높이=높이, 큐브 주행)는 이미 레퍼런스와 동일하므로 갈아엎지 않는다. "느낌"의 정체는
①스무딩→블록, ②발광→무광, ③스크롤→무대, ④넓은지형→좁은트랙 네 가지뿐이며, 토글 비교로
정량적 취향 선별이 가능하다(ADR 0009 옵션화 정신 계승).

## 대안
- 무채색 강제 전환: 색은 본질이 아님 → 기각(성부색 유지, 옵션).
- 전면 재구축: 스탠스가 이미 맞으므로 불필요 → 토글식 점진 개선 채택.
