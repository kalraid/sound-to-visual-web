# ADR 0005 — 분류·이조 캐논 감지

## 맥락
곡을 캐논 / 화음 / 기타로 구분. 캐논은 키가 달라도(이조) 성립. 자동 감지는 휴리스틱이라 오분류 가능.

## 결정
- **기타**: 단성(동시 발음 거의 없음, 성부 1개).
- **화음**: 다성(여러 성부 동시 발음 많음).
- **캐논**: 절대음 아닌 **음정 간격열(연속 음정차)** 을 만들어, 한 성부를 여러 lag로 밀어 다른
  성부와 비교 → 특정 lag에서 일치율 높으면 캐논(이조 허용).
- **결과**: 라벨 + 신뢰도(0~1). 애매하면 화음/기타 폴백.
- **수동 변경**: 프론트 드롭다운으로 사용자가 라벨 교체 가능.
- **견고성**: 파싱 실패/빈 곡/미지원 형식은 명확한 에러.

## 근거
음정 간격열 비교는 조옮김에 불변 → 키 다른 캐논도 감지. 휴리스틱 한계는 수동 변경으로 보완.

## 대안
- 절대음 동음 비교: 이조 캐논 놓침 → 기각.
- ML 분류기: 데이터·복잡도 과함 → 휴리스틱으로 충분.
