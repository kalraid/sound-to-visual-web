# ADR 0005 — 분류·이조 캐논 감지

## 맥락
곡을 캐논 / 화음 / 기타로 구분. 캐논은 키가 달라도(이조) 성립. 자동 감지는 휴리스틱이라 오분류 가능.

## 결정
- **기타**: 단성(동시 발음 거의 없음, 성부 1개).
- **화음**: 다성(여러 성부 동시 발음 많음).
- **캐논**: 절대음 아닌 **음정 간격열(연속 음정차)** 을 만들어, 한 성부를 여러 lag로 밀어 다른
  성부와 비교 → 특정 lag에서 일치율 높으면 캐논(이조 허용).
- **결과**: 라벨 + 신뢰도(0~1). 애매하면 화음/기타 폴백.
- **수동 변경**: 프론트 드롭다운으로 사용자가 라벨 교체 가능.
- **견고성**: 파싱 실패/빈 곡/미지원 형식은 명확한 에러.

## 보정 (C5, 2026-06-22)
- **토널 응답 허용**: 조성 답句(tonal answer)는 일부 음정이 1반음 달라져 정확 일치만으로는
  캐논을 놓치고 '화음'으로 오분류됐다. → `_similarity(tol=1)`(1반음 허용) 일치율이
  `_TONAL_MIN_RATIO`(0.78) 이상이고 **실제 시간지연 lagSec ≥ 0.3s** 이면 캐논으로 인정.
  시간지연 게이트로 동시발음(화음/코랄, lag≈0)의 거짓양성을 막는다.
- `classify`는 `canon.detected`(토널 보정 포함)를 1차 기준으로 사용.
- **수동 변경 구현**: 툴바 `category-select` 드롭다운으로 라벨 교체(약속만 있던 기능).
  캐논으로 바꾸면 추격 강조 on, 비캐논으로 바꾸면 off(`terrain.setCanonEmphasis`).
- 회귀 확인: melody→기타, canon→캐논, bwv66(코랄)→화음 유지(테스트 4종 통과).

## 근거
음정 간격열 비교는 조옮김에 불변 → 키 다른 캐논도 감지. 토널 응답·시간지연 게이트로
정밀도와 재현율을 함께 높이고, 남는 휴리스틱 한계는 수동 변경으로 보완.

## 대안
- 절대음 동음 비교: 이조 캐논 놓침 → 기각.
- ML 분류기: 데이터·복잡도 과함 → 휴리스틱으로 충분.
