# ADR 0004 — 색·시각효과·악보

> **개정(ADR 0011):** 렌더 스타일 토글 추가 — 기존 `glow`(블룸+발광) 외에 `matte`(블룸 끔,
> 무광 재질+soft shadow+중립 배경). 성부색은 유지. 악보 패널은 OSMD/2D 피아노롤 선택형(grill 확정).

## 맥락
사용자가 도~시 음계 개념을 언급. NHK 영상의 시각적 화려함도 목표.

## 결정
- **색**: 레인 = 성부색, 지형 표면/음표 = 음계(피치클래스 도~시) 색. 같은 '도'는 어느 성부든 같은 색.
- **타격 효과**: 새 음 시작마다 큐브 발광 + 파티클(소리와 동기).
- **악보**: OSMD 전체 렌더 + 커서 위치로 가로 자동 스크롤.

## 근거
피치클래스 색으로 화음·조성 구조가 색으로 드러남(사용자 도~시 개념과 직결). 발광/파티클로 영상미↑.

## 대안
- 성부색만 / 음계색만: 한쪽 정보 손실 → 둘 다 채택(레인=성부, 표면=음계).
- 악보 전체 축소: 긴 곡 가독성 나쁨 → 가로 스크롤.
