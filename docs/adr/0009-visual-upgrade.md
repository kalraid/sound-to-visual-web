# ADR 0009 — 비주얼 고도화 (Phase 1.4, MVP 피드백 반영)

## 맥락
MVP는 동작하나 (1) 악보가 가로 스크롤이라 "지금 어디?"가 불명확, (2) 3D가 얇은 선 2가닥
이라 NHK/유튜브의 공감각(부피감·발광·반응)이 부족, (3) 큐브가 위치마다 음계색으로 바뀌어 혼란.

## 결정
- **악보(ADR 0004 갱신)**: 가로 스크롤 → **현재 한 줄(시스템)만 크게, 진행 시 다음 줄로 페이지 넘김**(세로 follow).
- **큐브/지형 색(ADR 0004 갱신)**: 위치별 음계색 폐기 → **성부마다 고정색**(유튜브처럼). 큐브=지형=같은 성부색.
- **지형 형태(ADR 0003 갱신)**: 얇은 튜브 → **채워진 입체 지형**(음높이 라인 아래를 벽/언덕으로 채움) + 발광.
- **후처리/효과**: 블룸(발광 번짐) + 반사 그리드 바닥 + 비트 반응.
- **사용자 선택 옵션(3택 드롭다운)** — 돌려보고 불필요한 것 추후 제거:
  - **카메라**: ① 높은 3/4 부감(전체 조망) ② 낮은 추격(레이싱) ③ 측면 스크롤.
  - **배경/바닥**: ① 어두운 그라데이션 + 반사 그리드 ② 우주/별 ③ 단색.
  - **비트 반응**: ① 드럼/강박 ② 모든 음 시작 ③ 없음.

## 근거
"한 줄 페이지 넘김"이 현재 위치 가독성에 최선. 입체 지형 + 블룸 + 비트반응으로 공감각.
취향 의존이 큰 카메라/배경/반응은 옵션화해 사용자가 실사용으로 선별.

## 대안
- 여러 줄 + 세로 스크롤: 맥락은 좋으나 "한 줄 집중" 요구와 어긋남 → 페이지 넘김 채택.
- 한 가지 고정 연출: 취향 편차 큼 → 3택 옵션.
