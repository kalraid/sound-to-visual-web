# 용어집 (Glossary)

- **성부 (voice / part)**: 동시에 진행되는 독립 선율 줄. music21 `part`/`voice`로 추출.
- **캐논 (canon)**: 한 선율을 다른 성부가 시간차로 모방하는 형식. 성부 간 시간지연 자기유사성으로 감지(이조 허용 → 음정 간격열 비교).
- **화음곡 (harmonic / polyphonic)**: 여러 성부가 동시에 울리지만 캐논 구조는 아닌 곡.
- **기타 (melody / other)**: 단성(한 줄) 또는 위 둘에 속하지 않는 곡.
- **AMT (자동 채보)**: 오디오 → 음표 변환. 어렵고 손실 큼 → Phase 2.
- **단일소스 (source of truth)**: 모든 렌더가 참조하는 하나의 데이터/시계 = 백엔드 분석 JSON + `Tone.Transport`.
- **레인 (lane)**: 성부별로 분리된 지형 트랙(z축으로 나란히 배치).
- **리본 (ribbon)**: 음높이를 보간한 부드러운 지형 메쉬.
- **피치클래스 (pitch class)**: 옥타브 무시한 12음계(도~시 = 0~11). 색 매핑에 사용.
- **importance**: 성부 중요도 점수 = 음표밀도 + 외성(최상/최하) 가중 + 주제선율 보유(캐논).
- **몇분음표**: 음표 길이. music21 `quarterLength`(4분음표=1.0) ↔ 템포로 초 변환.
- **리듬 트랙**: 음정 없는 타악기(드럼) 트랙. 지형 맨 아래 레인에 비트 펄스로 표시.
