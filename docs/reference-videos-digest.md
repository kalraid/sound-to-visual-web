## 영상 1 — 캐논 변주곡의 시각화 (NHK 명곡앨범플러스)

**채널:** 헤세드 | **길이:** 5:00 | **곡:** 파헬벨 캐논 | **출처:** NHK 명곡앨범플러스
**URL:** https://www.youtube.com/watch?v=5Q-c7BS1wpM
**프레임:** `docs/reference-frames/video1-pachelbel/` (3초 간격 캡처 중 대표 8장)

### 실제 화면 (5초 간격 관찰 결과)
- **무채색 점토/건축 모형 디오라마 월드.** 흰색·회색 무광 재질, 부드러운 그림자, 하늘엔 작은 구름.
  화려한 색·발광이 전혀 없다. (`v1_000s` 타이틀 — 회색 하늘 아래 흰 블록 섬 3개)
- **번호가 매겨진 정사각 "섬/구역"(1→10+)이 흰 리본 길로 연결**되어 이어진다. 각 섬 = 곡의 한 구간/변주.
  카메라가 섬에서 섬으로 길을 따라 이동.
- **캐논의 시차 모방을 명확히 연출:** 빨강·초록·파랑 **3개 큐브(=3성부)가 출발대에 정렬**해 있다가
  (`v1_015s`, "Canon in D" 라벨), **빨강이 먼저 계단을 내려가고 초록·파랑이 시차를 두고 뒤따른다**
  (`v1_020s` 빨강만 하강 → `v1_035s` 초록·파랑 시차 출발). **같은 구조를 시차로 추격 = 캐논.**
- **각 섬은 음악 구조를 3D 조형으로 빚은 것.** 구간마다 형태가 완전히 달라진다:
  - 격자 계단/미로 (`v1_035s`, `v1_065s`)
  - 솟아오른 막대 숲 = 음높이 막대그래프 (`v1_100s`)
  - **거대한 링(도넛) + 나선 홈** = 캐논의 순환 화성 진행을 원으로 (`v1_130s`)
  - U자 스위치백 리본 미로 (`v1_160s`)
  - 흑백 블록 부조(밝은/어두운 블록 혼합) (`v1_185s`)
  - 평행 사다리 가로대 = 빠른 음들이 레일처럼 (`v1_205s`)
  - 계단식 산맥 + 봉우리에 컬러 깃발 마커 (`v1_230s`)
  - 흰 파이프 롤러코스터 루프 = 큐브가 관을 타고 흐름 (`v1_285s`)
- **작은 컬러 큐브(초록·빨강·파랑·보라)가 "성부=주자".** 구조물 위 길을 미끄러져 이동하고,
  봉우리에선 깃발 마커로도 표시된다.
- **카메라:** 디오라마 위를 부드럽게 비행/패닝하며 큐브를 따라간다.

### TL;DR
파헬벨 캐논을 **무채색 건축 디오라마들의 연속**으로 만들고, **컬러 큐브(성부)가 같은 길을
시차로 통과**하게 해 캐논의 모방 구조를 보여준다.

## 영상 2 — 14 canons / J.S.Bach

**채널:** onishikeita | **길이:** 4:56 | **곡:** 바흐 「14개의 카논 BWV1087」 중 7곡 | **출처:** NHK 명곡앨범플러스
**URL:** https://www.youtube.com/watch?v=mdI7UM7Xubw
**프레임:** `docs/reference-frames/video2-bach/` (대표 9장) + `video2-bach/3s/` (3초 정밀 12장, 2026-06-22)

### 실제 화면 (5초 간격 관찰 결과)
- **영상1과 같은 무채색 미니멀 3D**(흰/회색 무광, 아이소메트릭). 배경은 균일한 회색.
- **오프닝:** 주제를 `A B C D E F G H / 1 2 3 4 5 6 7 8` 라벨 큐브 한 줄로 제시 (`v2_000s`) —
  "주제 = 8개 음의 시퀀스"를 명시하고 시작. 각 카논은 이 주제의 변형임을 암시.
- **각 카논(No.1, 2, 3 … No.13)이 독립 장면.** 좌하단에 **2D 피아노롤(컬러 음표 막대)**을
  항상 띄워 실제 악보를 기준 자료로 병치 (`v2_030s` 이후 전 구간).
- **핵심: 회문·거울 대칭의 3D 시각화.** 흰색 블록 구조 **아래에 검은색 거울상**이 함께 나타나고,
  **양쪽에 같은 컬러 마커가 동시에 움직인다** — 흰=원 성부, 검=반사(역행/전위)된 성부.
  (`v2_060s`, `v2_090s` — 위 흰 블록 + 아래 검은 거울 블록, 노랑 마커가 양쪽 동시 주행)
- **방/구덩이 구조:** 흰 벽 + 검은 바닥 블록으로 대칭 공간을 만들고 마커 2개가 통과 (`v2_165s` No.9).
- **클라이맥스(No.13):** **두 벽면 + 바닥으로 된 3D 코너**에서, 한 벽엔 흰 부조·다른 벽엔
  검은(또는 윤곽선) 거울상·바닥엔 또 다른 면 — **직교 3평면 반사로 회문/거울을 입체 구현**
  (`v2_250s`, `v2_285s`). 청록·빨강·노랑 하트 마커가 각 면에서 대응 이동.

### Notable Quote (제작자 설명문)
> "visualized the mechanism of these pieces which can be compared to palindromes and mirrors"
> (회문과 거울에 비유되는 곡의 작동 원리를 시각화했다)

### 3초 정밀 재분석 (2026-06-22, 99장 추출 → 핵심 12장 직접 관찰, `video2-bach/3s/`)
기존 관찰을 확정하고 **구현에 직결되는 디테일**을 추가로 확인:
- **거울상의 반사축 = 바닥면(y=0) 아래로 내림.** 흰 원본 구조 바로 **밑에 검은 거울상**이
  수면 반사처럼 매달린다(`v2_048s` No.2, `v2_072s`·`v2_099s` No.3). → 우리 C4는 옆 레인으로
  반사 중이라 **방향이 틀림**: 아래(y 반전)로 내려야 레퍼런스와 일치.
- **각 성부가 자기만의 가느다란 리본 길**을 따로 통과한다. 평평한 병렬 레인이 아니라
  **XZ 평면에서 굽이치는 얇은 경로**(계단·고리·지그재그). 화음/성부가 한 줄에 뭉치지 않고
  **각자 독립 경로**로 간다(`v2_120s` No.5, `v2_201s` No.11의 굽은 흰 리본).
- **카논마다 완전히 다른 독립 조형물 + 타이틀 카드("No.1"…"No.13") + 장면 전환.** 길이 균등분할이
  아니라 곡(악장) 단위로 장면이 바뀌고 카메라가 그 조형에 맞춰 프레이밍.
- **No.13 클라이맥스: 직교 3평면(바닥+벽2) 반사.** 한 면은 흰 부조, 다른 면은 검은/윤곽 거울상
  (`v2_249s`, `v2_288s`). 마커는 청록·빨강·노랑 작은 콘/큐브.
- 좌하단 2D 피아노롤은 **상시 작은 오버레이**(토글 아님). 성부 수가 늘면 롤도 2단으로 분할 표시.

### TL;DR
바흐 14 canons를 **흰 구조 + 검은 거울상**으로 빚어, 각 카논의 **회문·거울(역행/전위) 대칭 메커니즘**을
3D 반사로 보여준다. 2D 피아노롤을 기준 자료로 병치.

## 비교 (Comparison)

| 항목 | 영상1 (파헬벨) | 영상2 (바흐 14 canons) |
|------|----------------|------------------------|
| 미감 | 무채색 점토 디오라마 | 무채색 미니멀 3D + 거울상 |
| 구조 표현 | 섬(구역) 연속 + 길 | 카논별 독립 장면 + 거울 대칭 |
| 성부 | 컬러 큐브가 길을 시차 통과 | 컬러 구/하트가 구조 통과 |
| 캐논 강조 | **같은 길 시차 추격** | **흰 구조 ↔ 검은 거울상** |
| 보조 자료 | 없음 | 2D 피아노롤 상시 표시 |

### 두 영상이 공유하는 설계 언어 (공통 DNA)
1. **무채색 무광 3D(흰/회색).** 오직 **움직이는 성부 마커만 컬러** — 나머지는 전부 흰/회색.
2. **음악의 "구조"를 3D 건축/조형으로 빚는다.** 음표를 흘리는 게 아니라, 곡의 설계도를 입체로 만든다.
3. **성부 = 그 구조를 통과하는 작은 컬러 마커.** 캐논은 "같은 구조를 시차로/거울로" 통과하는 모습.
4. **아이소메트릭~3/4 부감 + 부드러운 카메라 이동.** 차분하고 명상적인 톤.
